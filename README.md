# ChatFlow

WebFlux 기반 실시간 채팅 서비스

## 프로젝트 소개

본 프로젝트는 실시간 채팅 서비스를 구현하며  
WebFlux, WebSocket, Redis Pub/Sub, Kafka 등의 기술을 활용해  
이벤트 기반 아키텍처와 대규모 메시지 처리 구조를 학습하기 위한 개인 프로젝트입니다.

## 목표

- 실시간 메시징 시스템 구현
- Reactive Programming(WebFlux) 실전 적용
- Redis 기반 메시지 브로드캐스팅 구조 설계
- Kafka 기반 이벤트 구조 확장 학습
- Docker 및 CI/CD, AWS 배포 경험 확보

## 기술 스택

- Java 21
- Spring Boot
- Spring WebFlux
- Spring Security + JWT
- WebSocket
- Redis (Pub/Sub)
- MySQL
- Docker (예정)
- GitHub Actions (예정)
- AWS (예정)
- Kafka (예정)

## 구현 기능 (MVP)

### 1단계
- 회원가입
- 로그인 (JWT)
- 인증 API

### 2단계
- 채팅방 생성
- 1:1 채팅
- 그룹 채팅
